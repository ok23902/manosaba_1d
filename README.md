# 魔法少女ノ魔女裁判 1Dプロトタイプ

まのさば非公式二次創作。AIを用いた簡単なテキストアドベンチャーゲーム。

## 必要なもの
- Python 3.10以上
- DeepSeek、ChatGPT、Claude、ローカルLLMのAPIキー
※参考：DeepSeekのAPIサイト→https://platform.deepseek.com
※DeepSeek以外で試してないのでベータ機能です。ご注意ください。

## 動かし方
1. .envファイルを作成
2. LLM_PROVIDER=your_llm_providerを記入（デフォルトはdeepseek）
3. LLM_API_KEY=your_key_here を記入
4. pip install -r requirements.txt
5. python game.py
